PinHammer for Hammer

A lightweight Hammer template pack using the advantages of Hammer to help kick off site build projects.

No Frameworks

Thats right, there are no Frameworks involved in this set of Hammer templates. This is kept simple so that you can use it as a basis for creating your own designs and layouts or quickly add the frameworks you need. We have put in place the following:

  • Normalised CSS
  • REM font sizing
  • Golden ration modular type scale
  • ARIA roles
  • Basic mixins to help with a responsive design approach - depending on your design change them to suit the page layout for the different screen sizes - break them where your layout needs to break not at common screen sizes.
  • Commented link in the code to sources of interest
  • CSS styling page with a range of common html tags to help with styling
